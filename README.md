# udfonts-adder

Add universal design fonts to Microsoft Office for mac as user-defined fonts.

Microsoft Office for Macのユーザ定義フォントにモリサワBIZ UDゴシック、BIZ UD明朝を追加する。

## Usage

```
% git clone https://github.com/xight/udfonts-adder
% cd udfonts-adder
% ./install.sh
```
